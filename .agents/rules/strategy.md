# Style-alyst: Agentic AI Personal Stylist

## Business Requirements

- An MVP of a chatbot application functioning as a high-end boutique personal stylist ("Style-alyst")
- The chatbot should be integrated on Product Detail Pages (PDP) from www.jcpenney.com
- Example PDP URL context: `https://www.jcpenney.com/p/ana-womens-crew-neck-sleeveless-vest/ppr5008658749?pTmplType=regular&deptId=dept20000013&catId=cat100210006&rrplacementtype=home1_jrecs&criteriaName=E0_S4_MV_MostViewed`
- **Visual Input**: Users can upload a photo of a celebrity look, a Pinterest "vibe", or an item from their closet.
- **Multimodal Analysis**: Use Vision-LLMs to analyze colors, textures, and silhouettes in the photo.
- **Agentic Search**: Autonomously browse JCPenney's vast catalog to find the closest matching products.
- **Complete the Look**: Build a cohesive, complete look (Top + Bottom + Shoes + Accessory) fitting user's budget and size.
- **Contextual Logic**: Understand "Occasions" (e.g., "Make this look appropriate for a summer wedding").
- **Goals**: Target 25% - 35% Average Order Value (AOV) lift, increase cross-category attachment rates, and mitigate return rates.
- The priority is a slick, professional, gorgeous UI/UX with intuitive features for the chatbot.

## Stunning Chatbot UI/UX Features

To ensure the "Style-alyst" feels like a premium, high-end boutique experience, implement the following design principles:
- **Glassmorphism & Depth**: Use frosted glass effects (`backdrop-filter: blur`), subtle drop shadows, and layered UI elements to create a modern, floating interface over the PDP.
- **Fluid Micro-Interactions**: Incorporate smooth, spring-based animations for opening the chat, sending messages, and expanding product carousels (e.g., using Framer Motion).
- **Rich Media Messages**: Display outfit recommendations in beautifully styled, horizontal scrollable carousels with high-quality JCPenney product images, clear pricing, and a "Shop the Look" button.
- **Typographic Hierarchy**: Use elegant, modern sans-serif fonts with carefully balanced weights to mimic fashion editorial layouts.
- **Skeleton Loaders**: Replace generic spinners with skeleton screens that mimic the shape of outfit carousels while the agent is "searching" or "analyzing" the user's uploaded photo.
- **Contextual Prompts**: The chat widget button should subtly pulse and show a contextual prompt (e.g., "Need styling advice for this vest?") based on the specific PDP the user is currently viewing.

## Technical Details

- Implemented as a modern React 18 app, client-rendered
- The React 18 app should be created in a subdirectory `frontend`
- Integration with JCPenney Product Image API and real-time inventory (mocked if necessary for MVP)
- Requires Vector databases for "Visual Search" and LLMs for fashion reasoning (e.g., GPT-4o or Gemini Pro Vision)
- No persistence and no user management for the MVP
- Use popular, modern libraries
- As simple as possible but with an elegant UI

## Strategy

The execution of the MVP (Phase 1: Closet Matcher) is broken down into a rigorous 10-part execution plan (detailed in `plan.md`):
1. **Parts 1-3 (Planning & Foundations)**: Establish the architecture, scaffold the React 18 app, and build the Mock JCPenney PDP background.
2. **Parts 4-7 (Core Widget & Mock APIs)**: Develop the stunning Chatbot UI widget and connect it to mock data services for the initial user flows.
3. **Parts 8-10 (AI Integration & Polish)**: Connect Vision-LLMs, implement the Agentic "Complete the Look" logic, and apply final UI polish (rich media carousels).
4. **Testing & QA**: Carry out extensive integration testing, fixing defects. Ensure a "high-touch" personalized service feel.
5. **Completion**: Only complete when the MVP is finished and tested, with the server running and ready for the user.
